LBANet: the clone war
-----------------------------------------------------


__________________
Installation:
1- launch the game "LBANet.exe"

2- enter any nickname/ password when asked

3- enter the world you want to play in

 

__________________
In game command:


arrow keys: move twinsen
mouse wheel: zoom in/out
mouse right button: change camera view

F1: normal mode
F2: sportive mode
F3: aggresive mode
F4: discrete mode
F5: horse
F6: dino-fly
F7: protopack

F10: reset twinsen position on the map, use it if you are stucked
F12:change from windowed mode to fullscreen

PageUp/PageDown: to be used with dino-fly

ESC: game menu
